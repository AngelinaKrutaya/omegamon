1. Review 'LoopSOAP.bat' to make sure that the correct input argument values, especially for CURL_path and SOAP_Request_Dir, are specified.  

2. Run 'LoopSOAP.bat' to start sending the SOAP requests, which are stored in the 'request' subdirectory. 

3. The file 'SOAP_OUTPUT_MMDDhhmmss.txt' is produced to contain the SOAP responses.

4. Run 'Summarize_SOAP_output.bat' to get a summary of the latest 'SOAP_OUTPUT_MMDDhhmmss.txt'.
